Tiptoe Company Website Project

A responsive frontend development project for NSW TAFE IT Level 4


The Tiptoe company website was developed during my NSW TAFE IT Level 4 studies, focusing on responsive web design and advanced JavaScript features. This project showcases my frontend development skills in building a modern, mobile-friendly business website.

I implemented key interactive elements including animated slideshows and comprehensive form validation to enhance user experience and accessibility. The website is fully responsive, adapting seamlessly across desktop, tablet, and mobile devices.

This project reinforced my mastery of core frontend technologies such as HTML5, CSS3, and JavaScript, while addressing real-world business requirements. The JavaScript components challenged me to create smooth animations and robust validation systems, ensuring both aesthetic appeal and functional reliability.


Features

  • Responsive Design: Fully adaptive layout for all device sizes
  • Interactive Slideshows: JavaScript-powered image carousels
  • Form Validation: Client-side input validation for contact forms
  • Mobile Navigation: Hamburger menu for small screens
  • Smooth Animations: CSS and JavaScript transition effects
  • Cross-browser Compatibility: Consistent performance across browsers

Tools Used

  • Frontend: HTML5, CSS3, JavaScript
  • Animation: JavaScript (vanilla), CSS transitions
  • Development Tools: VS Code, Chrome DevTools
  • Design: Photoshop/PhotoPea, Tiniy
